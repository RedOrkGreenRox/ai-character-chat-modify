## What Workshop stores

Cloudflare D1 stores:

- Discord user id, username, and avatar URL, used for identity and moderation.
- If you link GitHub: GitHub login, GitHub id, and an encrypted OAuth token with `gist` scope. The token is used only to create/delete Gists on your behalf when you publish or delete your own content.
- Published item metadata: titles, tags, descriptions, ratings, report counters, content URLs.
- Votes, installs, reports, moderation status.
- Session tokens for login.

## What Workshop does not store

- The actual published content. Content lives in GitHub Gists under the author's GitHub account.
- Discord email or server list; only `identify` scope is requested.
- Raw IP addresses in the application database. Cloudflare may process request logs according to Cloudflare's own policies.

## Removing data

Contact support@<your-domain>, or use the in-app delete account action when available.

Deleting your Workshop account removes Workshop metadata. It does not automatically remove your GitHub Gists unless you delete them yourself or use a feature that explicitly deletes the Gist with your linked GitHub token.
