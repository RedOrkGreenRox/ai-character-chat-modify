# ACCM Mini Library

Small import-ready sample library for local testing.

## Folders

- `lorebooks/` — SillyTavern/WorldInfo-style JSON lorebooks. Can be published as `kind: lorebook` or imported by Workshop lorebook installer.
- `skillbooks/` — Markdown skillbooks. Can be published as `kind: skillbook`; currently installs as lore context until dedicated Skillbook activation UI exists.
- `characters/` — simple AI Character Chat-compatible character JSON drafts. Can be published as `kind: character`.
- `workshop-items/` — ready-to-copy Workshop publish payloads that wrap the same files with metadata.

The files are intentionally small so you can publish/import them one by one.
