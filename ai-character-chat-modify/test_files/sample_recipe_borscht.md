# Борщ — краткая тестовая памятка

Классический борщ обычно готовят из свёклы, капусты, картофеля, моркови, лука, томатной пасты и бульона.

Один из приёмов: свёклу лучше тушить отдельно с кислотой — лимонным соком или уксусом — чтобы цвет оставался ярким.

Подача: со сметаной, чесноком, зеленью и чёрным хлебом.
